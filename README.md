# peli-projekti
Hankelähtöinen tuotekehitysprojekti, jonka tavoitteena on suunnitella ja toteuttaa toimiva pelikonsepti ideasta valmiiksi prototyypiksi.

Projektissa sovelletaan tuotekehityksen ja pelisuunnittelun periaatteita käytännön työskentelyssä. Työhön sisältyy pelisuunnittelu, ohjelmointi ja grafiikkasisältöjen tuottaminen sekä versionhallinnan hyödyntäminen.

Tavoitteena on kehittää pelattava demoversio, joka havainnollistaa pelimekaniikan ja visuaalisen ilmeen keskeiset elementit. Projektin aikana harjoitellaan tiimityöskentelyä, projektinhallintaa ja ketterän kehityksen menetelmiä.
